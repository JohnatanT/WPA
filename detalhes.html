Aplicativo de Anotações

1. Esta aplicação foi construida no curso de Progressive Web App realizado pela School of Net, com o obejtivo de abordar os principais conceitos e dar uma introdução aos alunos sobre o mesmo.


2. Algumas das vantagens de usar PWA:

3. Menor fricção: como é baseado na web, é eliminada a jornada clássica para uso de um app nativo (ir para a loja, buscar o app, clicar, esperar instalar e, somente então, usar o app).


4. Alcance: tem um alcance potencialmente maior, com esforços de desenvolvimento menores.


5. Descoberta (SEO-friendly): maior facilidade de indexação e de se encontrar todo conteúdo de um PWA.


6. Linkabilidade: qualquer pagina tem um link direto, o que facilita o compartilhamento.


7. Sempre renovado: não é necessário acessar a app stores para baixar últimas versões.


8. Responsivo: funciona nas diversas telas (smartphones, tablets e desktops).


9. Uso Offline: graças ao Service Workers pode funconar até mesmo sem conexão, uma vez depois de acessado.


10. Notificações: ferramenta chave para engajamento, é possível usar notificações em Android, mesmo sem estar acessando o PWA
